Computational Analysis Tools
Version 1.0

Objective: 
  A library of capabilities from which most common types of computation
  analysis applications can be rapidly assembled using System 3 as the
  underlying framework and the enhancements of the Native compiler to
  improve productivity.

The Cat software library is released under the GNU Public License for
Libraries.  Cat is an extensible library written for Native System 3
(see ETH's Native Oberon web page for pertinent licensing and copyrights:
http://www.oberon.ethz.ch/native/).
