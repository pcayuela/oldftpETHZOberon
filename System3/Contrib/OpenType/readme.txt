This directory contains the latest release of OType, a package for
using TrueType fonts in (Native) Oberon System 3. Files ending in
.Arc are Oberon archives created with the Oberon Compress utility.

Files

OType045.Arc      current release
OType045.tar.gz   current release in gzipped tar file

Recent Changes
22.06.2000 - release 0.45
- introduced cvtLen in OType.Font to deal with fonts that have no CVT
10.12.1999 - release 0.44
- fix in OTInt.Skip: EIF may immediately follow IF
- assert valid memory access in OTFonts.FillRect

These files are also in the Native Oberon release.
