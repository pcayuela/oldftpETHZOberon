ETH Oberon R2.32 (MacOberon S3R2.32) beta version. 25.5.2000
Comments/Corrections to <gs@statlab.uni-heidelberg.de> please.

The full source files are in Src.Arc and must be expanded with Compress.
The Mpp.Arc file contains some sources in Mpp format. Using in Builder.Compile
wiht the /D options, object files for Windows or PPC can be built directly 
from these merged sources.

Restrictions/Limitations:

This is a PowerMac version only.
Support for more than 8bit colors is still experimental.
MacX not yet supported.
