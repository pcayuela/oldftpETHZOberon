Welcome to the world of Oberon!
-------------------------------

This is the root directory for all Oberon implementations provided by ETH 
Zurich and some public domain versions of commercial Oberon-related tools.
Sources from various Oberon books can also be found here. 

  Books/       contains sources from various Oberon books.
  Newsletter/  archive of the Oberon Newsletters (incomplete).
  BlackBox/    educational version of BlackBox (previously Oberon/F).
  OberonV4/    ETH Oberon V4 (older version).
  Ofront/      evaluation version of an Oberon-2 to C translator.
  System3/     ETH Oberon System 3, an evolution of Oberon with a GUI.
  ETHOberon/   Link to latest version of ETH Oberon (System3).

Please see the readme.txt files for further information.  You can get help
at news:comp.lang.oberon or mailto:oberon@inf.ethz.ch.  The ETH Oberon home
page is at: http://www.oberon.ethz.ch/
