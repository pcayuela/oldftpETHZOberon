XE - Extended Edit	SHML	7 Mar 96	ludwig@inf.ethz.ch
supports folded text, compilation according to file-extension, default
file-extensions, configurable menus, and several keyboard shortcuts.
Also, a simple Macro processing module is included.

In order for AsciiCoder.DecodeFiles not to overwrite your
original XE.Menu.Text and XE.Configuration.Text, two *.def files
are extracted which you can inspect, and, if you like them, rename to the
required file names.

XE.Open XE.Menu.Text.def
XE.Open XE.Configuration.Text.def

System.RenameFiles
	XE.Menu.Text.def => XE.Menu.Text
	XE.Configuration.Text.def => XE.Configuration.Text
~

Changes since 8 Jan 96 (see also XE.Mod):
  17 Jan 96: XE.Comp of a list of files will stop after first error,
    open the file, and mark the errors in it
  31 Jan 96: selected text remains selected when an UpdateMsg with
    id Texts.replace is sent (e.g. change font no longer clears selection).
  2 Feb 96: restructured configuration of menu frame, changed command names
  14 Feb 96: XE.Compile <filelist> generates compiler command from file
    extension, InNameSet/TrackSelection accept "@" as part of a name
    -> e-mail addresses
  7 Mar 96: SHML - double left click clears selection. Useful for
    Edit.Search, when something is changed and the selection remains, but
    it's not the searchable selection.
